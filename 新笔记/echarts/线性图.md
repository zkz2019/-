### 记录一些折线图使用经验

- 配置类
  有一些动态的配置,在初始化时写了可能会无效,可以尝试一下在改变后通过 option 链式赋值

  - tooltip 弹出框

    - 可以配置显示隐藏/背景色/边框/编辑/字体样式等
    - axisPointer 可以配置坐标轴指示器,就是移入图表时显示的轴线或阴影
    - formatter 自定义弹窗元素
      ```
      // 示例   添加百分比符号
      formatter:function (params) {
        let html = params[0].name + '<br>'
        for (let i = 0; i < params.length; i++) {
          html +=
            '<span style="display:inline-block;margin-right:5px;border-radius:10px;width:10px;height:10px;background-color:' +
            params[i].color +
            ';"></span>'
          html += params[i].seriesName + ':' + params[i].value + '%<br>'
        }
        return html
      }
      ```

  - lengend 图例

    - 可以配置位置,字体等

  - grid 绘图网格

    - 可以配置绘图区域大小/是否根据坐标轴来设置大小等

  - xAxis X 轴配置

    - 可以配置轴线显示/隐藏/刻度/样式;
    - X 轴的数据也在这里设置

  - yAxis Y 轴配置

    - 和 xAxis 基本一样
    - splitLine 分割线,可以设置线条样式等
    - interval 坐标轴分割间隔
    - max 坐标轴最大刻度
    - min 坐标轴最小刻度

  - series 设置数据
    - 如果设置了 type 是 'category'，但没有设置 axis.data，则 axis.data 的内容会自动从 series.data 中获取
    - 如果是双轴则将 yAxisIndex 设置为不同的值;如果 Y 轴的配置不同则在 yAxis 设置为有两个对象元素的数组

- 效果实现记录
  - 实现固定 y 轴 5 个刻度并且不出现小数
    - 拿到每个类目的最大值,获得整体的最大值
    - 将 max 设置为能被 5 整除的数(比如 13 就设置为 15,18 就设置为 20)
      ```
           max + 5 - (max % 5)
      ```
    - 将 interval 设置为 max 的 5 分之 1
      ```
          (max + 5 - (max % 5))/5
      ```
