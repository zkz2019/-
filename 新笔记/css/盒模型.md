### 在对文档进行布局时，浏览器的渲染引擎根据标准的 CSS 基本框模型将每个元素表示为一个矩形框

### 矩形框可以分成以下几部分

- 内容框
- 填充框
- 边框
- 边距框

### 当内容对于它所在的盒子来说太大时，我们称之为溢出。您可以使用该 overflow 属性管理元素如何处理溢出内容。

### 当 `box-sizing` 设置为 `content-box` 时, `width` 和 `height` 只能控制到内容区的大小,盒子的大小还会在设置尺寸的大小上再加上其他框的尺寸

### 当 `box-sizing` 设置为 `border-box` 时, `width` 和 `height` 会将内容框/填充框/边框的整体尺寸控制在设置的尺寸,也就是会根据 border 和 padding 来设置 content 的尺寸(content = width - border.width \* 2 - padding \* 2);盒子的大小会在设置尺寸的大小上再加上边距框尺寸

### 当出现滚动条时,滚动条尺寸会加在填充框上;比如 Y 轴出现滚动条,则填充框的右侧会增加一个滚动条宽度,类似于 padding-right 加了一个滚动条宽度

### 轮廓和阴影占据的是边距框一起的位置,如果存在边距的话;但是他们两个不会增加盒子的尺寸,如果没有边距也不会将附近的盒子撑开
